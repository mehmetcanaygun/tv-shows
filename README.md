## TV Shows

Search and display TV Shows, see details and add them to your favourites. Built with Vue & TailwindCSS.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
